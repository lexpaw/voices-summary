
#### 功能简述
从录音笔设备中读取音频文件，识别音频内容、区分说话人，转为文字稿，并对文字进行摘要保存。

#### 用户需求描述

1. 软件以 Windows 系统后台程序形式运行，通过任务栏托盘图标打开控制台。
2. 根据指定USB录音笔独有的设备特征，当录音笔插入电脑USB接口时，自动识别出设备连接。
3. 检测到录音笔接入后，自动从录音笔设备存储空间（会映射为磁盘）读取音频文件列表，判断哪些音频文件未经过同步。
4. 将未同步的音频文件复制到指定数据目录。
5. 调用语音识别服务，将音频文件转写为带说话人标记的文本，保存到指定输出目录。
6. 调用大语言模型服务对音频转写内容进行总结，提炼出会话的各个要点，保存到指定输出目录。
7. 提供用户界面，实现音频播放，完整对话预览，会话摘要查看等功能。
8. 技术栈使用 Rust + Tauri，前端使用 Typescript，使用 Vue 框架，前端管理工具使用 Bun。
